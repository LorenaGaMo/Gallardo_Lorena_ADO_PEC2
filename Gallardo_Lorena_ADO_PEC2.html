ABSTRACT

En esta PEC se quiere hacer un anlisis de datos RNA-seq de treinta muestras a partir de un estudio con muestras de clulas del tejido tiroidal, a las que se les ha aplicado diferentes tratamientos. Estas treinta muestras estn divididas en tres grupos de diez que corresponde a los tres tratamientos de infiltracin que se llev a cabo. Con el anlisis de datos se buscaron las diferencias significativas de la expresin gnica de los diferentes grupos.

OBJETIVOS

El objetivo de este trabajo es comparar los niveles de expresin gnica de estos tres grupos de muestras de tejido tiroidal dos a dos para obtener un listado de genes que se expresan de manera diferencial tanto los sobre expresados como subexpresados. En este caso haremos un experimento de comparacin de clases. Las tres comparaciones posibles: - NIT vs SFI: Para comparar el efecto del tratamiento SFI con no tratados. - NIT vs ELI: Para comparar el efecto del tratamiento ELI con no tratados. - ELI vs SFI: Para comparar los dos tratamientos.

Software y naturaleza de los datos

El software utilizado para realizar este estudio es el siguiente: R 1 (versin 3.6.1) con el interfaz RStudio 2 y los paquetes de Bioconductor Affy 3 (versin 3.10).
